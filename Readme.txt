1. If you are newly installing the admin and web please only use the Admin and web New Install zip file, 
	You don't need to use the Admin and web Update to version-x zip file. 
	The Admin Update zip file is only for update the older version.
2. The update is available for previous two version which means you can't update the system by the update file from any older version. If you need to update from very older version kindly contact our support or install the latest version as a fresh copy. 
3. If you need only the changed files for mobile apps in V16.1 check the apps changed files from V16.0 to V16.1, otherwise please use User app and Vendor app zip/folder.
